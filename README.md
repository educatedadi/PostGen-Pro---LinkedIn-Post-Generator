# PostGen Pro

Generate high-engagement LinkedIn posts with AI. Built with Vite, React, TypeScript, and Supabase.

## Getting started

Requirements:
- Node.js (LTS) and npm

Install and run locally:

```sh
# Install dependencies
npm install

# Start the dev server
npm run dev
```

## Tech stack
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Environment variables

Create a `.env` file based on your Supabase project settings:
- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_PUBLISHABLE_KEY`
- `VITE_SUPABASE_PROJECT_ID`
